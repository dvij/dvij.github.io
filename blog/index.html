Finding the Best Nash Equilibrium in Polynomial Time (For a Meaningful Class of Games)

optimization

Nonconvex QPs on the simplex are NP-hard in general. But a reparameterization argument shows that a large, natural class of potential games — where intrinsic payoffs are strong relative to strategic interactions — admits exact, polynomial-time global optimization. The key is a certification SDP with only n+1 constraints.

Finding Nash equilibria is hard. Finding the best Nash equilibrium — the one maximizing social welfare — is harder. For a natural and well-studied class of games, this is an NP-hard optimization problem even when the game is fully described and finite.

This post describes a new result that carves out a meaningful tractable class: potential games where the intrinsic strategy payoffs are strong enough relative to the strategic interactions. For these games, the welfare-maximizing Nash equilibrium can be found in polynomial time via a certify-and-solve pipeline. The certification requires solving an SDP with just n+1 constraints.

The Setting: Potential Games

A potential game is one where every player’s incentive to change strategy is captured by a single global function V : \Delta_n \to \mathbb{R} called the potential. Nash equilibria are exactly the KKT points of maximizing V over the simplex \Delta_n = \{x \geq 0 : \mathbf{1}^\top x = 1\}.

The simplest nontrivial potential is quadratic:

V(x) = x^\top A x + q^\top x, \qquad x \in \Delta_n

Here A_{ij} is the payoff interaction between strategies i and j, and q_i is the intrinsic payoff to strategy i — present regardless of what others play. The gradient \nabla_i V(x) = 2(Ax)_i + q_i is the marginal payoff to strategy i given population state x, so KKT on the simplex says exactly: at a Nash equilibrium, all active strategies have equal (and maximal) payoff.

When A \succeq 0, V is concave on the simplex, it has a unique global maximum, and any local solver finds it. Welfare maximization is trivial.

When A is indefinite, V can have multiple local maxima — multiple stable Nash equilibria. Finding the global maximum (the welfare-best equilibrium) is NP-hard in general. Standard approaches: copositive relaxations for bounds, or global solvers like BARON for small instances. No polynomial-time exact method was known for any nontrivial indefinite class.

Until now (modulo the caveat that this hasn’t been peer-reviewed yet).

A Concrete Example: Rock-Paper-Scissors with Intrinsic Rewards

Consider a 3-strategy symmetric game with interaction matrix

A = \begin{pmatrix} 0 & -1 & 1 \\ 1 & 0 & -1 \\ -1 & 1 & 0 \end{pmatrix}

— pure Rock-Paper-Scissors, where each strategy beats one and loses to the other. This is skew-symmetric, so A + A^\top = 0; the unique Nash equilibrium is the uniform distribution x^* = (1/3, 1/3, 1/3), and the potential x^\top A x = 0 everywhere (since A is skew-symmetric). The potential landscape is flat — every point on the simplex is a Nash equilibrium with equal welfare. This is a degenerate, uninteresting case.

Now add intrinsic payoffs q = (2, 1, 0): strategy 1 (Rock) has an intrinsic advantage — perhaps it’s less cognitively costly, or more culturally salient. The potential becomes:

V(x) = x^\top A x + 2x_1 + x_2

This is indefinite (since A is skew-symmetric, A + A^\top = 0 \not\succeq 0), so naively NP-hard. But let’s check if it’s in our tractable class.

Step 1: Check convexifiability. We want c \geq 0 such that

M(v_k) = A + \text{diag}(c \circ \nabla V(v_k)) \succeq 0

at each of the 4 vertices of \mathcal{S} = \{x \geq 0 : \mathbf{1}^\top x \leq 1\}:

  • v_0 = (0,0,0): \nabla V(v_0) = q = (2, 1, 0)
  • v_1 = (1,0,0): \nabla V(v_1) = 2Ae_1 + q = (-2, 2, -2) + (2,1,0) = (0, 3, -2)
  • v_2 = (0,1,0): \nabla V(v_2) = 2Ae_2 + q = (2, -2, 2) + (2,1,0) = (-... )

Wait — at v_1 we get \nabla_3 V = -2 < 0. With c \geq 0 required, the correction c_3 \cdot (-2) \leq 0 can’t help M(v_1) \succeq 0 in the (3,3) entry. The SDP is infeasible for this particular q.

That’s informative. It says this specific perturbation of RPS is not in the tractable class — the intrinsic advantage to Rock isn’t strong enough to convexify globally. The gradient of V is negative in the third component at v_1, which the c \geq 0 constraint can’t correct.

Step 2: Strengthen the intrinsic payoffs. Try q = (5, 5, 5) — all strategies have a strong, equal intrinsic reward. Now \nabla V(v_0) = (5,5,5) \gg 0, and at each vertex the large q dominates the 2Ax term. The SDP becomes feasible, and the reparameterization certifies global tractability.

The game-theoretic interpretation: when all strategies have a strong reason to be played independently of the opponent’s action, the equilibrium structure simplifies — the potential landscape has a unique global maximum that the reparameterization finds exactly.

The General Result

The setup: minimize -V(x) = -x^\top A x - q^\top x (equivalently, maximize V) over

\mathcal{S} = \{x \geq 0 : \mathbf{1}^\top x \leq R\}.

A componentwise reparameterization x_i = s_i(y_i) with curvature \kappa_i = s_i''/{s_i'}^2 = c_i transforms the Hessian of the objective to

M(x) = -A + \text{diag}(c \circ (-\nabla V(x))) \succeq 0.

For -\nabla V(x) > 0 (i.e., the potential is locally decreasing — we’re minimizing) and c \geq 0, the diagonal correction adds positive terms to -A, potentially pushing it PSD.

The key structural fact. M(x) is affine in x, so \lambda_{\min}(M(x)) is concave on \mathcal{S}. A concave function on a simplex attains its minimum at a vertex. Therefore:

Note

Theorem (Vertex Certification). The reparameterization convexifies -V over all of \mathcal{S} if and only if it does so at the n+1 vertices v_0 = 0,\; v_i = Re_i. This reduces to an SDP with n+1 LMI constraints: \text{find}\; c \geq 0 \;\text{s.t.}\; -A + \text{diag}(c \circ (-\nabla V(v_k))) \succeq 0, \quad k = 0, \ldots, n. \tag{SDP}

For a box [0,R]^n, the analogous argument requires 2^n vertices. The simplex geometry is what makes this polynomial.

The pipeline:

  1. Certify: solve (SDP). If feasible, the reparameterization x_i = -\frac{1}{c_i}\log(1 - c_i y_i) convexifies the problem globally.
  2. Solve: minimize the (now convex) reparameterized objective over the convex y-domain using any first-order method.
  3. Recover: x^* = s(y^*) is the global welfare-maximizing Nash equilibrium.

What Does the Tractable Class Look Like?

The SDP is feasible — roughly — when the linear term q (intrinsic payoffs) is large and positive relative to the indefiniteness of A (strategic interactions). More precisely:

  • At v_0 = 0, the condition reduces to -A + \text{diag}(c \circ q) \succeq 0, which requires q > 0 and c large enough to dominate \lambda_{\max}(-A).
  • At v_i = Re_i, the condition involves -\nabla V(Re_i) = -2RA_{:,i} - q, which requires q + 2RA_{:,i} not to be too negative.

The tractable class is a convex set in (A, q) space (being the feasible set of an SDP). It includes:

  • Any game with A \preceq 0 (concave potential) — trivially, since -A \succeq 0 already.
  • Games where \|q\|_\infty is large relative to \|A\| — the weak interaction regime.
  • Games where A has only small negative eigenvalues — slight indefiniteness, easily corrected.

What’s excluded: games where the strategic interactions are the dominant force (\|A\| \gg \|q\|), and in particular homogeneous games with q = 0. The Motzkin-Straus problem (finding the maximum clique via \max x^\top A_G x on the simplex with q=0) is provably outside the tractable class for all nontrivial graphs, which is reassuring since max clique is NP-hard.

The Phase Transition

The numerical picture sharpens this. On 500 random indefinite potential games (n=4, A indefinite, q \sim 3\mathcal{N}(0,I)):

Outcome Count Fraction
Certifiably convexifiable 411 82%
— gradient sign-changing (nontrivial) 220 54% of certifiable
— interior global optimum 29 13% of nontrivial
Not certifiable 89 18%

The 29 instances with interior optima are the most significant: the welfare-maximizing equilibrium is in the interior of the simplex, not at a pure strategy. Vertex enumeration would miss these entirely. The reparameterization pipeline finds them exactly.

As \|q\|/\|A\| increases, the certifiable fraction approaches 100%. There’s a sharp transition — empirically near \|q\|_\infty / |\lambda_{\min}(A)| \approx 2R — between the tractable and intractable regimes. Characterizing this boundary analytically is an open problem.

The Dynamical Interpretation

There’s a beautiful connection to evolutionary dynamics that makes the result feel deeper.

The replicator equation for a potential game with V(x) = x^\top A x + q^\top x is:

\dot{x}_i = x_i\bigl(\pi_i(x) - \bar\pi(x)\bigr), \qquad \pi_i(x) = (Ax)_i + q_i, \quad \bar\pi = x^\top \pi(x)

This is the gradient flow of V under the Shahshahani metric on the simplex interior. Replicator dynamics in potential games always converge to Nash equilibria — but generically to local maxima of V, not necessarily the global one.

In the certifiable class, V (after reparameterization) has a unique global maximum. This means:

Important

Corollary (Dynamical Selection). For potential games in the certifiable class, the replicator dynamics select the welfare-maximizing Nash equilibrium from almost all initial conditions, and this equilibrium can be computed in polynomial time.

This combines two things that are usually in tension: the equilibrium is simultaneously dynamically stable (selected by the natural learning dynamics) and computationally tractable (findable in polynomial time). Outside the certifiable class, you can have multiple stable equilibria with different welfare properties — the learning dynamics may select a suboptimal one, and finding the optimal one is hard.

Connection to Classical Tractability Results

The classical approach to tractable QCQPs works via the dual: find Lagrange multipliers that convexify the Lagrangian (the S-lemma for one constraint, Burer & Ye’s “at most one negative eigenvalue” condition as the sharpest known unifying result). Chordal sparsity allows decomposition across variables.

The reparameterization approach is primal and complementary: change variables so the objective is convex, certify this via vertex checking on the simplex. The two approaches likely capture overlapping but non-identical tractable classes — understanding their precise relationship is an open problem.

What’s clear is that the simplex geometry is doing real work. The n+1 vertex structure is what makes the certification polynomial. For a box domain, the same idea requires 2^n LMI constraints and fails to scale. The simplex — the natural domain for mixed strategies in game theory and probability simplices in other applications — is exactly the right domain for this technique.

Open Questions

A few things I don’t know the answer to:

Is there a closed-form boundary? The certifiable class is an SDP-defined convex set in (A, q) space. Is there a simpler characterization — e.g., certifiable if and only if \lambda_{\min}(A) \geq -f(q, R) for some explicit f?

Does the tractable class extend classical results? Are there instances certifiable by reparameterization but not by Lagrangian convexification (joint definiteness / Burer-Ye), or vice versa?

Branch-and-bound on sub-simplices. For games outside the certifiable class, the simplex can be subdivided. On smaller sub-simplices, the gradient is more likely to have consistent sign, and the SDP becomes feasible. Can this give a B&B algorithm that terminates quickly in practice even for hard instances?

Rates for the replicator dynamics. In the certifiable class, the reparameterized potential is strictly concave after the change of variables. Does this imply polynomial convergence rates for the replicator equation (under the reparameterized metric)?

Appendix: Why n+1 Vertices?

The simplex \mathcal{S} = \{x \geq 0 : \mathbf{1}^\top x \leq R\} has exactly n+1 vertices: the origin v_0 = 0 and the n scaled basis vectors v_i = Re_i. The matrix M(x) = -A + \text{diag}(c \circ (-\nabla V(x))) is affine in x (since \nabla V(x) = 2Ax + q is affine). The minimum eigenvalue of an affine matrix pencil is concave. A concave function over a simplex attains its minimum at a vertex. Therefore checking M(v_k) \succeq 0 at all n+1 vertices is necessary and sufficient for M(x) \succeq 0 everywhere on \mathcal{S}. No approximation is involved. \square
