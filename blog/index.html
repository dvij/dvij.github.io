Convexification of Simplex-Constrained QPs, with an Application to Potential Games

optimization

Optimizing an indefinite quadratic over the simplex is NP-hard. But a componentwise change of variables can render both the objective and the constraint convex, and whether it works is checkable by a small SDP with n+1 constraints.

The problem

Consider the quadratic program on the (full-dimensional) simplex:

\min_{x \in \Delta}\; V(x) = \tfrac{1}{2}\, x^\top Q\, x + q^\top x, \qquad \Delta = \{x \in \mathbb{R}^n : x \geq 0,\; \mathbf{1}^\top x \leq 1\}.

Here Q \in \mathbb{R}^{n \times n} is symmetric and \Delta has n+1 vertices: the origin and the n standard basis vectors.

  • If Q \succeq 0, V is convex — gradient descent finds the global minimum.
  • If Q \preceq 0, V is concave — the minimum is at a vertex; check n+1 points.
  • If Q is indefinite, neither approach works. The problem is NP-hard in general.

This post describes a method that handles a large subclass of the indefinite case. A componentwise change of variables x_i = s_i(y_i) can render both the objective and the constraint convex, and whether such a change exists is checkable by a semidefinite program with only n+1 constraints.

The reparameterization

We ask: is there a smooth, componentwise map x_i = s_i(y_i) that makes V(s(y)) convex in y?

The Hessian

For maps s_i with constant curvature \kappa_i = s_i''/(s_i')^2, the Hessian of V(s(y)) factors as:

H(y) = S'(y) \cdot M(x) \cdot S'(y)

where S' = \text{diag}(s_1', \ldots, s_n') is positive and:

M(x) = Q + \text{diag}\!\big(\kappa \circ \nabla V(x)\big), \qquad \nabla V(x) = Qx + q.

Since S' \succ 0, convexity of V(s(y)) is equivalent to M(x) \succeq 0 for all x \in \Delta.

Both objective and constraint become convex

For convex maps s_i (the case \kappa_i > 0):

  • Objective: V(s(y)) is convex (by the PSD condition on M).
  • Constraint: since s_i(0) = 0 and s_i is increasing, x_i \geq 0 is equivalent to y_i \geq 0. The remaining constraint \sum_i x_i \leq 1 becomes \sum_i s_i(y_i) \leq 1 — the sublevel set of a convex function. The feasible set \{y \geq 0 : \sum_i s_i(y_i) \leq 1\} is the intersection of the nonneg orthant with this sublevel set — convex.

The full reparameterized problem is: minimize a convex function over a convex set — solvable in polynomial time by projected gradient descent.

Note

Maximization for free. Convexification also solves \max V on \Delta: a convex function attains its maximum at a vertex of a polytope, so we just evaluate V at the n+1 vertices. The same certificate handles both directions.

Reducing to a finite check

Two key facts:

  1. M(x) is affine in x (because \nabla V is), hence matrix-concave.
  2. A matrix-concave function on a polytope attains its minimum eigenvalue at a vertex.

\Delta has n + 1 vertices. So checking M(x) \succeq 0 everywhere reduces to:

\text{find}\; \kappa \;\;\text{s.t.}\;\; M(v_k) \succeq 0, \quad k = 0, 1, \ldots, n \tag{SDP}

This is a semidefinite program with n+1 matrix constraints of size n \times n — polynomial time.

Note

Why the simplex matters. On a box [0,1]^n, the same argument requires 2^n vertex checks. The simplex has only n+1.

Matrix-concavity is what matters. The vertex trick requires M(x) to be matrix-concave — meaning each diagonal entry M_{ii}(x) must be concave as a function of the full vector x, not just x_i. With constant \kappa_i, M_{ii}(x) = Q_{ii} + \kappa_i (Q_i^\top x + q_i) is affine in x and therefore concave. Allowing \kappa_i to depend on x_i makes M_{ii} quadratic in x, and the cross-term Hessian \partial^2 M_{ii}/\partial x_j \partial x_k = b_i(Q_{ij}\delta_{ik} + Q_{ik}\delta_{ij}) is indefinite whenever Q has off-diagonal entries. Finding reparameterization families richer than constant curvature while preserving matrix-concavity remains open.

The tractable class

The tractable class \mathcal{T} is the set of (Q, q) pairs for which the SDP is feasible. For a given (Q, q), the gradients g_k = Qv_k + q are fixed vectors, and the SDP is an LMI in \kappa — solvable in polynomial time.

Certifiability depends on the sign pattern of the gradients across vertices. Each \kappa_i > 0 adds \kappa_i g_{k,i} to the i-th diagonal of M(v_k): a positive g_{k,i} gives leverage to boost Q_{ii} past the off-diagonal entries, while a negative g_{k,i} fights against it. If g_i > 0 at every vertex, coordinate i can be boosted arbitrarily — but this is the trivial case where the optimum is at the origin. The interesting regime is when gradients have sign conflicts across vertices: increasing \kappa_i helps at some vertices and hurts at others, and certifiability reduces to whether the tradeoff can be balanced.

Limits

A necessary condition. Reparameterization is a diffeomorphism — it preserves all critical points and their Morse index (Milnor 1963, Lemma 2.2). After convexification, V \circ s is convex, so every critical point has index 0 (is a local minimum) — in particular, there are no saddle points. This means the method can only succeed when V has a unique critical point on \Delta: not just a unique local minimum, but no saddle points at all.

The inclusion chain. Let \mathcal{U} denote the set of (Q, q) for which V has a unique critical point on \Delta. Then:

\{Q \succeq 0\} \;\subset\; \mathcal{T} \;\subset\; \mathcal{U} \;\subset\; \{\text{all QPs on } \Delta\}.

Both inclusions are strict. What makes \mathcal{T} useful is the complexity gap:

  • Testing \mathcal{U} requires verifying that no KKT point exists on any of the 2^n - 1 nonempty faces of \Delta other than the global minimum — exponential time.
  • Testing \mathcal{T} is a single LMI in \kappa — polynomial time.

So \mathcal{T} is a tractable inner approximation of \mathcal{U}. The gap \mathcal{U} \setminus \mathcal{T} consists of QPs with a unique critical point whose gradient sign pattern across vertices cannot be balanced by any \kappa > 0.

Application: welfare maximization in potential games

Background

A potential game on n+1 strategies has a potential V(x) whose critical points on the population simplex are Nash equilibria. After coordinate elimination, the reduced potential is a QP on \Delta \subset \mathbb{R}^n.

The payoff to strategy i is F_i(x) = \partial V / \partial x_i. Social welfare is W(x) = x^\top F(x) = x^\top (Qx + q) — itself a QP on \Delta, so the convexification framework applies directly to W. Finding the welfare-maximizing equilibrium is NP-hard in general. For certifiable instances, \max W is at a vertex.

Note

Potential \neq welfare. The potential is V = \tfrac{1}{2}x^\top Q x + q^\top x; welfare is W = x^\top Q x + q^\top x = 2V - q^\top x. They differ by a linear term. The framework applies to either — they are both QPs on \Delta — but welfare is the economically meaningful objective.

When is welfare indefinite?

In pure congestion games, each strategy’s payoff decreases with adoption: Q_{ij} \leq 0 for all i, j. Then Q is negative semidefinite, welfare is concave, and the maximum is at a vertex — trivially solvable.

Indefiniteness requires positive cross-effects: adopting strategy j raises the value of strategy i (network externalities, ecosystem synergies, complementary infrastructure). These positive off-diagonal entries in Q compete with negative self-congestion on the diagonal, creating indefiniteness. Platform ecosystems, technology adoption, and multi-modal transport networks naturally exhibit this structure.

Example 1: platform ecosystem (vertex optimum)

Three platforms compete for a unit mass of users. The payoff to platform i is F_i(x) = \sum_j Q_{ij} x_j + q_i, where:

Q = \begin{pmatrix} 1 & 0 & 0.5 \\ 0 & 0.5 & 1 \\ 0.5 & 1 & 1 \end{pmatrix}, \qquad q = \begin{pmatrix} 8 \\ 6 \\ 2 \end{pmatrix}.

Platform 1 (premium) has the highest intrinsic value (q_1 = 8) and moderate network effects. Platform 3 (basic) has low intrinsic value but strong cross-platform synergies (Q_{23} = 1, Q_{13} = 0.5) — it benefits heavily from the ecosystem. Q is indefinite (eigenvalues \approx -0.35, 0.90, 1.95) because the cross-platform effects are large relative to the self-effects.

Welfare is W(x) = x^\top(Qx + q) on \{x \geq 0, \sum x_i = 1\}. Eliminating x_3 = 1 - x_1 - x_2 gives the reduced welfare QP on \Delta = \{x \geq 0, \mathbf{1}^\top x \leq 1\} \subset \mathbb{R}^2:

\tilde{Q} = \begin{pmatrix} 2 & -1 \\ -1 & -1 \end{pmatrix}, \qquad \tilde{q} = \begin{pmatrix} 5 \\ 4 \end{pmatrix}.

\tilde{Q} is indefinite (\det = -3). Certification: with \kappa = (0.4,\; 0.7), M(v) \succ 0 at all three vertices. Since W is convexifiable, the maximum is at a vertex:

W(1,0,0) = 9, \qquad W(0,1,0) = 6.5, \qquad W(0,0,1) = 3.

The welfare-maximizing equilibrium concentrates all users on platform 1 (premium). Despite the ecosystem synergies, the intrinsic quality gap is decisive. The certificate guarantees no mixed adoption achieves higher welfare.

Example 2: platform adoption (mixed optimum via concavification)

Not all welfare functions are convexifiable. Some admit concavification instead — M \preceq 0 with \kappa < 0, making W \circ s concave. This certifies a unique global maximum directly, which need not be at a vertex.

A different platform game has stronger cross-platform synergies. The reduced welfare QP:

\tilde{Q} = \begin{pmatrix} -4 & 4 \\ 4 & -1 \end{pmatrix}, \qquad \tilde{q} = \begin{pmatrix} 2 \\ 4 \end{pmatrix}.

\tilde{Q} is indefinite (\det = -12): the large positive off-diagonal (Q_{12} = 4, cross-platform synergy) dominates the negative diagonal (self-congestion). This welfare function is NOT convexifiable, but it IS concavifiable with \kappa < 0.

x^* \approx (0.23,\; 0.77,\; 0), \qquad W(x^*) \approx 17.85.

Users split between platforms 1 and 2 (favoring platform 2). The synergy between platforms creates a mixed optimum that is strictly better than any pure strategy — the vertex maximum is W(0,1,0) = 17.5. The concavification certificate guarantees x^* is the unique global maximum.

Numerical evidence

On 500 random indefinite QPs on \Delta (n = 4, \|q\|/\|Q\| \approx 3, vector \kappa via SDP):

Count %
Unique critical point (\mathcal{U}) 454 91%
Certifiably tractable (\mathcal{T}) 78 16%

\mathcal{T} covers roughly 10% of \mathcal{U}: most indefinite QPs with a unique critical point cannot be certified by constant-curvature reparameterization. The gap reflects sign conflicts in the gradient pattern that no \kappa > 0 can resolve. Closing this gap — via variable-curvature reparameterizations or SOS certificates — is the main open problem.

Dynamics

In potential games, learning dynamics (replicator, gradient play) converge to some Nash equilibrium — but not necessarily the welfare-maximizing one. When the welfare QP is certifiable, the reparameterized W has a unique global maximum, so:

Note

Corollary. For certifiable welfare QPs, gradient-based dynamics in the reparameterized space converge to the welfare-maximizing equilibrium from almost all initial conditions.

Open questions

A diagonal dominance sufficient condition. A simpler sufficient condition for M(x) \succeq 0 comes from diagonal dominance: \kappa_i \cdot g_i(x) + Q_{ii} \geq \sum_{j \neq i} |Q_{ij}| at all vertices. Is there a scaled version (à la Geršgorin) that is tight?

Richer reparameterizations. Constant-curvature maps give M(x) affine — hence matrix-concave, enabling the vertex trick. The natural generalization \kappa_i(x_i) makes M_{ii} quadratic with indefinite Hessian, destroying matrix-concavity. Can a different parameterization extend the tractable class?

Relation to the S-lemma. The classical S-lemma establishes joint convexity of the numerical range of a pair of quadratic forms. Reparameterization convexifies the primal objective directly. Do these certify the same class?
